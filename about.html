Flask Framework

Flask is a lightweight WSGI web application framework in Python. It's designed to be simple and easy to use, allowing developers to start small and scale up to complex applications.

Key Features:
  • Built-in development server and debugger
  • RESTful request dispatching
  • Jinja2 templating
  • Secure cookies support
  • Unit testing support
SQLAlchemy

SQLAlchemy is a Python SQL toolkit and Object Relational Mapper (ORM) that gives developers the full power and flexibility of SQL.

Key Features:
  • Object Relational Mapping (ORM)
  • Schema Migration Tools
  • Database Agnostic
  • Connection Pooling
  • Powerful Query Builder
Bootstrap

Bootstrap is the world's most popular framework for building responsive, mobile-first sites.

Key Features:
  • Responsive Grid System
  • Pre-styled Components
  • JavaScript Plugins
  • Customizable Themes
  • Cross-browser Compatibility
SQLite

SQLite is a C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.

Key Features:
  • Serverless Architecture
  • Zero Configuration
  • Self-contained
  • Highly Reliable
  • ACID Compliant
